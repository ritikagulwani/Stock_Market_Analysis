## Getting Started
To run this project locally, follow these steps:

1.Clone the repository

2.Install the required libraries and q yfinance library for dataset

3.Launch Google Colab: https://colab.research.google.com/

4.Open the Stock_py.ipynb file and run the notebook cells sequentially.
